# Pharmaceutical Sales Analytics Dashboard – Power BI

## Project Overview

This project is an interactive Power BI dashboard developed to analyze pharmaceutical sales data across multiple time dimensions including hourly, daily, weekly, and monthly trends. The dashboard provides detailed business insights into medicine category performance, sales patterns, demand fluctuations and time-based purchasing behavior.

The main objective of this project is to transform raw pharmaceutical sales data into meaningful visual insights that can help businesses, analysts and decision-makers understand product demand and optimize operational strategies.

# Business Problem

Pharmaceutical businesses generate large volumes of sales data every day. However, raw transactional data alone cannot provide actionable insights.

This project addresses key business questions such as:

- Which medicine categories generate the highest sales?
- How do sales vary across days, weeks, and months?
- At what time does customer demand peak?
- Which product categories show seasonal trends?
- How can sales patterns improve inventory planning and business decisions?

The dashboard converts complex datasets into easy-to-understand visual analytics.

# Project Objectives

- Analyze pharmaceutical sales performance
- Identify top-performing drug categories
- Track hourly, daily, weekly, and monthly sales trends
- Monitor demand fluctuations over time
- Enable data-driven decision-making
- Improve sales and inventory understanding through visualization

# Dataset Information

The project uses four datasets representing different time granularities of pharmaceutical sales data.

## Datasets Used

| Dataset | Description |
| `saleshourly.csv` | Hourly sales records |
| `salesdaily.csv` | Daily sales records |
| `salesweekly.csv` | Weekly aggregated sales |
| `salesmonthly.csv` | Monthly aggregated sales |

# Features Present in Dataset

The datasets contain sales values for multiple pharmaceutical product categories:

| Feature | Description |
| `M01AB` | Anti-inflammatory and antirheumatic products |
| `M01AE` | Anti-inflammatory products (Propionic acid derivatives) |
| `N02BA` | Analgesics and pain relief medicines |
| `N02BE` | Aniline analgesics and antipyretics |
| `N05B` | Anxiolytic drugs |
| `N05C` | Hypnotics and sedatives |
| `R03` | Drugs for obstructive airway diseases |
| `R06` | Antihistamines for systemic use |

Additional temporal features include:

- Date
- Year
- Month
- Hour
- Weekday Name

# Dashboard Insights

Some major insights obtained from the dashboard include:

- Certain pharmaceutical categories consistently outperform others in sales.
- Hourly data reveals peak purchasing periods during specific times of the day.
- Monthly trends indicate seasonal fluctuations in medicine demand.
- Weekly analysis helps identify recurring sales behavior patterns.
- Product demand varies significantly across different time intervals.
